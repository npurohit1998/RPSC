# RPSC 1st Grade Teacher Exam - Study PWA 📚

## 🚀 कैसे इस्तेमाल करें

### Step 1 - Files Setup
इन फाइलों को एक साथ रखें:
- `index.html` (मुख्य app)
- `manifest.json` (PWA config)
- `sw.js` (service worker)

### Step 2 - Browser में खोलें
किसी भी modern browser में `index.html` खोलें।

### Step 3 - API Key डालें
पहली बार खुलने पर Google Gemini API Key डालें।
Free key मिलती है: https://aistudio.google.com/app/apikey

### Step 4 - Mobile पर Install करें
Browser में "होम स्क्रीन पर जोड़ें" option से install करें।

## 📱 Features

### 🏠 होम
- परीक्षा काउंटडाउन (16 मई 2026)
- आज का study goal
- Progress stats
- Quick actions
- Top important topics

### 📋 टॉपिक
- सभी 20+ विषयों के माइक्रो टॉपिक
- Importance rating (1-5 ⭐)
- पिछले 15 papers का औसत
- Search और filter

### 🎯 क्विज़
- Gemini AI से हिंदी में प्रश्न
- Custom subject और question count
- गलत उत्तर से auto flashcard
- Bookmark difficult questions
- विस्तृत explanation

### 📝 फ्लैशकार्ड
- गलत उत्तरों से automatic बनते हैं
- Flip animation
- Easy/Hard tracking
- Spaced repetition

### 📊 प्रगति
- Subject-wise progress
- Overall accuracy
- Study time tracking
- Personal notes
